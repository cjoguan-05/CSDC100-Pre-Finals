## Project Title
GenAI Guide: FAQs and Ethical Use of AI

## Short Overview
This project is a simple, student-friendly website that explains what Generative AI is, how it works, and how to use it responsibly in school and daily life. It includes a FAQ page, an ethics page with checklists, and an analogy page that connects AI to real development work. The site is built using HTML, CSS, and Bootstrap, and is easy to view in any browser.


> **Note:** On GitHub, the repository root itself is the *CSDC100-Pre-Finals*.
